## Getting Started

Follow steps below to get up and running!

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/colorado-quinn/DozerMatch.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Run project
   ```sh
   npm run dev
   ```
